Projects

Some things that may or may not be worked on.

The links on this site currently link to my repositories. This will be changed in the future to feature custom webpages.


PXGA Cursors - Yet another pixel cursor set, 'cause clearly, there aren't enough.

RegInfoMod - A small Windows program to change the registered user and organization, displayed in places like winver.exe.

Project Daisy (PC Version) - A game of currently unspecified variety for computers.

Sonic 1 (SGDK Remake) - An attempt at recreating (some of) Sonic the Hedgehog 1 in the Sega Genesis Development Kit.

Project Daisy (SEGA Genesis Version) - A game of currently unspecified variety for the SEGA Genesis. This repository is currently archived.

Project Blaze - A game project for the SEGA Genesis that failed miserably, and is not worth checking out. This repository is currently archived.

Super Mario Bros. for the SEGA Genesis - A recreation of the original SMB for the SEGA Genesis using the Sega Genesis Development Kit.

SVGI Cursors - A cursor set generated from custom-drawn vector files. They are (in my opinion) not as high quality as my later PXGA Cursor Set.
